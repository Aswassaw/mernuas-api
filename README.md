# Mernuas
Api untuk Mernuas, sebuah aplikasi berbasis MERN Stack dengan studi kasus sebuah authentikasi luar biasa, yaitu login dengan Google dan/atau Github.

## Teknologi yang digunakan
- ExpressJS
- Mongoose
- JWT
- Nodemailer
- Google Auth Library
- Bcrypt
- Morgan
